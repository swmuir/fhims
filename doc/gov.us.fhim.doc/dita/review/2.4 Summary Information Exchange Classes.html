BehavioralHealthSummaryReport Class

This class specifies the summary report intended to exchange selected information relevant across specialties. It may not include the details of an assessment but it will contain many data elements that are based on the information collected through the assessment and generated from its processing.

This attribute is used to specify that the content of this clinical document is sensitive because it contains Behavioral Health information. This attribute corresponds to the Act.confidentialityCode in the HL7 Version 3 RIM but is constrained to specify sensitivity and for the purpose of data segmentation.
